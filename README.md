# Insatallation 
Install the Node Modules on ```backend``` and ```spacewatcher``` directory:
```
npm install
```

Get a API Key at [Weather API](https://www.weatherapi.com/).

Make a copy of the ```.env.example``` file and name it ```.env```. Place the API key inside ```API_KEY= ```.

# Running the Project

From the ```backend``` directory:
```
npm run server
```

From the ```spacewatcher``` directory:
```
npm run dev
```
